Install the global ionic cli : npm install -g @ionic/cli
Then initiate the ionic app : ionic start (app name) (template name)
The command i used is : ionic start reciclica-app sidemenu

Then i went github and i created the new project named Ryciclica app

Create the boot screen : ionic generate page pages/boot-screen

Create the component : ionic generate component components/loader

Ionic app start at : localhost:8100

Install the ngrx :: npm install @ngrx/store --save
